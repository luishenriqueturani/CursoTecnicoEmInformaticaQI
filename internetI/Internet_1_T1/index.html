HTML

       O HTML (Hypertext Markup Language) é a linguagem usada para se desenvolver páginas web, ela não é uma linguagem de programação, mas sim de marcação. Foi desenvolvido inicialmente para auxiliar na troca de informações da academia de físicos do CERN, por Tim Berners-Lee. A página inicialmente era completamente rudimentar e não tinha uma boa aparência, era com fundo preto e letras verdes e não tinha opções de customizar. O HTML é constituido por um conjunto de tags que marcam os elementos de uma página.

       Editores de HTML

       Há diversos softwares que podem ser utilizados para a edição de documentos do tipo HTML. Alguns fornecem opções avançadas para a criação de layouts, com opção de pré-visualização da página, autofechamento de tags, e diversas outras funcionalidades.

       Para o aprendizado de HTML, há dois editores mais simples que podem ser ótimas ferramentas, justamente por incentivarem o aprendizado das tags de forma mais limpa, sem adicionar uma maior complexidade nesse momento: o Aptana Studio e o Notepad++.
